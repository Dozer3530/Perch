# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Contributors and participants are expected to treat each other with respect.
Harassment of any kind is not tolerated.

## Reporting

If you experience or witness behaviour that violates this code, report it by
opening a private vulnerability advisory on GitHub at
https://github.com/Dozer3530/Perch/security/advisories/new (used as the
private contact channel since this repo is small).

Reports will be reviewed and acted on in good faith.
